# Frontend Core: Vigile Cieco Branding

## Visual Identity
- **Name**: Vigile Cieco (Blind-Shield)
- **Theme**: Cyber-Navy (#020617) with Electric Blue (#0ea5e9) accents.
- **Typography**: Inter (Weights: 300, 400, 600, 800).
- **Glassmorphism**: Backdrop blur (12px) with 40% opacity navy backgrounds for cards.

## Key Components
1. **Exposure Tool**:
   - Interactive scanner simulating AI exposure analysis.
   - Drag & Drop support for local files.
   - Step-by-step progress visualization (15% to 100%).
2. **Pricing Grid**:
   - Responsive cards for Monthly (1,90€) and Annual (19€) plans.
3. **Legal Checkout Modal**:
   - Gated payment access requiring 3 mandatory legal/marketing checkboxes.
   - Integration with PayPal SDK (Monthly) and SumUp (Annual).
4. **Intelligent Assistant**:
   - Fixed RAG-style chat widget.
   - Deterministic knowledge base for <10ms response time.
   - Persona: Serious, protective, and technical ("Vigile Cieco").

## Tech Stack
- **Framework**: Tailwind CSS (CDN).
- **Interactivity**: Vanilla JavaScript (ES6+).
- **Payment**: PayPal JavaScript SDK + SumUp External Links.
