# Manuale dell'Ispettore - AIGuard.io

## Missione
Garantire l'eccellenza tecnica e la coerenza del prodotto AIGuard.io.

## Protocollo di Verifica
1. **Auto-Test Obbligatori:** Nessun codice viene pushato senza il superamento dei test unitari e di integrazione.
2. **Coerenza Multilingua:** Verificare che ogni nuova stringa UI sia presente in tutti i file di traduzione (IT, EN, DE, FR, ES).
3. **Verifica Frontend:** Ogni modifica alla landing page deve essere validata visivamente (screenshot) per assicurare la perfezione del design Dark Mode.
4. **Performance:** Monitorare il carico CPU del core Rust. Reazione < 50ms è il requisito non negoziabile.
