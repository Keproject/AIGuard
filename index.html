Intelligence FAQ

Cos'è il Vigile Cieco (Blind-Shield) di AIGuard.io?

È un protocollo di isolamento deterministico che impedisce ai processi identificati come motori di addestramento AI di accedere ai file locali dell'utente.

Perché AIGuard è scritto in Rust?

Rust garantisce prestazioni a latenza zero (< 10ms) e sicurezza della memoria, essenziali per un monitoraggio di sistema che non rallenti il PC.

Come funziona il rilevamento watermark AI?

Il modulo Forensics analizza i metadati e i pattern bit-level per identificare contenuti generati o pronti per l'ingestione da parte di modelli IA.

Quali file vengono monitorati?

Il modulo Watcher si focalizza su documenti critici come .pdf, .docx, .pem, .kdbx e file di configurazione sensibili.

AIGuard.io rallenta il mio computer?

No, l'architettura è ottimizzata per un impatto CPU < 0.1%. La reattività è garantita dal linguaggio Rust e da Tokio runtime.

Cosa succede se viene rilevato un processo sospetto?

Il sistema cambia lo stato della Tray in Rosso e può essere configurato per bloccare immediatamente l'accesso al file da parte del processo incriminato.

È compatibile con Windows 11?

Sì, AIGuard.io è progettato specificamente per le moderne architetture Windows 10 e 11.

I miei dati vengono inviati ai server AIGuard?

No. Certificato 'NO LLM TRAINING'. L'analisi è 100% locale. Nessun file lascia mai il tuo dispositivo.

Come si installa il software?

Dopo l'attivazione riceverai un installer leggero. L'attivazione è istantanea tramite chiave licenza digitale.

Qual è la giurisdizione legale?

KEPROJECT S.R.L. ha sede in Italia. Il foro competente per ogni controversia è quello di Pordenone.
